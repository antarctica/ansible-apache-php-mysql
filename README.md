# Apache PHP MySQL (`apache-php-mysql`)

**Part of the BAS Ansible Role Collection (BARC)**

Configures Apache PHP module to use MySQL PDO driver

## Overview

* Restarts Apache to pickup PHP PDO MySQL driver
